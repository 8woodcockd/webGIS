Linux: basics

This session will take you through the basics of Linux, an operating system commonly used to run webservers.

You're advised to read through Part 1 first. After this, you should login to our pre-installed Linux system. Once you have an OS to play with, you should then read through Part 2 and 3, trying out some of the commands (you'll ne able to navigate round, just not use the commands demanding superuser access).

  • Part 1: Introduction to Operating Systems and Linux
  • Part 2: Working at the command line
  • Part 3: Basic admin
  • Log into the Dialog-Plus server.

More information

For more information on Linux, especially Linux administration, good starter books include Linux All-in-One For Dummiesand Linux: The Complete Reference, though almost any similar book covering installation and security will do. Also, there's a good basic Unix tutorial at Surrey University's Electronic Engineering dept.

Installing Linux

If you're interested in installing your own copy of Linux, we'll look at this later in the course when we look at OSGeo-Live.
