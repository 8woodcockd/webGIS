Linux: basics

This session will take you through the basics of Linux, an operating system commonly used to run webservers.

You're advised to read through Part 1 first. After this, you can either have a go at installing your own version of Linux, or you can login to a pre-installed Linux system during the practicals. Once you have an OS to play with, you should then read through Part 2 and 3.

  • Part 1: Introduction to Operating Systems and Linux
  • Installing OSGeo-Live or accessing the Dialog-Plus server (do either, or both! As you feel).
  • Part 2: Working at the command line
  • Part 3: Basic admin

More information

For more information on Linux, especially Linux administration, see the Recommended reading. Also, there's a good basic Unix tutorial at Surrey University's Electronic Engineering dept.
