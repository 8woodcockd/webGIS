Formatting

Manually re-formatting the data is an incredibly awkward and tedious process..

Fortunately, there are now some tools available which expedite the process of converting from Excel to JSON.

  • CSV Conversions
  • Code Beautify
  • JSFiddle
This converts CSV data - which can be easily exported from Excel - to a JSON format.

Have a play with doing this. When you are happy, move onto Geo-coding.
