Web-based mapping tutorials

Here's a few tutorials for taking your web-based mapping on.

  • Harvesting Tweets: this tutorial shows you how to harvest tweets using Python or Java.
  • Dynamic mapping Twitter with Node.js: this tutorial shows you how to map Tweets as they are sent on GoogleMaps, using Node.js on a server to compile the Tweets.
  • Mapping using GitHub: automatic mapping using the popular software repository.
  • Dynamic mapping with GitHub: not a great solution to dynamic community mapping, but if you're on a tight budget...
  • The leaflet package for online mapping in R: combining Leaflet and R
  • Mapping with R and Shiney: online mapping of data processed with R
  • App Inventor for Android: building maps with GPS in phones
  • Using the location of a browser: how to offer maps based on a user's location.
  • Creating a map using Fusion Tables: mapping from GoogleDocs.
  • Creating simple maps from Google Spreadsheets: push button mapping.
