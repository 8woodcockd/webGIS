Google Maps Assessed practical

The practical task for unit 4 involved the construction of a map showing the locations of Blue Plaques in Leeds, marking the locations of buildings with a significant history.

Assessed task

Using the Google Maps techniques that you have learnt, you should develop your own version of the Leeds Blue Plaques map. You can enhance the general appearance of the map and web page however you think appropriate, and can link to further sites if you wish.

  • The web page should be written for a general audience and should thus contain contextual information about the plaques and their use.
  • The site can use multiple web pages if you wish.
  • Data sources should be cited.
  • You should study the Google Maps JavaScript API reference material to learn more about options controlling the appearance of such features as markers and infowindows.

The assessed work counts for 20% of the overall module mark

Submission details

The assessment for this practical will be based on the web page/s that you create for the above task. A short document should be submitted via the VLE, which gives details of the URL of your map. You will also be asked to submit:

  • A copy of the HTML page(s) used for your map
  • A copy of any external JavaScript files used for your map
  • A copy of the style sheet used if relevant

The files should be included using the 'zip file upload' feature of Turnitin.
